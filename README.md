# Toll Wise Driver App
Toll Wise Driver, a project that aims to increase mobility efficiency in Indonesia, is rooted in statistical data which reveals the fact that in 2021, the Central Statistics Agency (BPS) recorded 143,797,227 motorized vehicles spread throughout the archipelago. This data highlights the significance of mobility in modern life, with motorized vehicles being one of the most common ways to travel. Therefore, this project aims to overcome mobility challenges by providing solutions that can benefit society at large, especially in the context of increasing toll road use.

# Benefit
The target in using this information system is of course especially the government, namely BPJT (Toll Road Regulatory Agency) in carrying out service to drivers on toll roads. Apart from the government, this application too used by toll road users, starting from 4-wheeled vehicle drivers etc. Where with this application drivers can easily access toll road services.
